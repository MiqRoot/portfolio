CASO 01 — SAÚDE PÚBLICA
Sistema de Distribuição de Insulina
HTML CSS JavaScript MySQL

CASO 02 — IA APLICADA
TriagemPRO — Triagem Inteligente de Documentos
HTML CSS JavaScript Python PostgreSQL API Google (IA)

Sistema voltado a profissionais da saúde, jurídicos e contábeis, pra agilizar o atendimento a clientes e organizar documentos com mais rapidez. A IA analisa cada documento recebido, verifica o grau de urgência, classifica por categoria e gera um resumo automático do conteúdo — o profissional já recebe o documento pré-organizado e sabe o que priorizar antes mesmo de abrir tudo manualmente.

META → redução estimada de ~40% no tempo de organização de documentos por atendimento*
*projeção interna, aguardando validação em uso real
